# DeepDocClassifier

Mit DeepDocClassifier_Tensorflow.ipynb lassen sich unter Einspeisung von 
verschieden großen Datensets Modelle initialisieren, trainieren und anschließend 
abspeichern. Dabei muss man das unten erwähte Datenset auf die gewünschte Trainingsgröße
reduzieren. Die jeweiligen Ordnerpfade müssen angepasst werden.


Mit Test_DeepDoc.ipynb lassen sich die erstellten Modelle laden und mit einem Testdatenset 
testen. Anschließend lässt sich die Confusion Matrix abbilden.

Das für beide Dateien benötigte Datenset Tobacco 3482, findet man hier:
https://www.kaggle.com/patrickaudriaz/tobacco3482jpg
